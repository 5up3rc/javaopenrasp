### How to use:

# Java Open Rasp

这是一个java rasp的验证性demo，已验证rasp的原理及实现

### 实现的保护点

1. 反序列化漏洞

2. Ognl表达式执行

3. ProcessBuilder log

## SQL注入

1. MySql注入保护

2. SqlServer注入保护

编译后，将"-javaagent:javaopenrasp.jar"添加至JVM的启动参数
